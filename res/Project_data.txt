# 🤖 프로젝트 수행 능력 분석 보고서

**지원자: 이준호 (질문하는 AI 엔지니어)**

이준호 지원자는 데이터 수집부터 전처리, 모델링, 서비스 구현까지 AI 프로젝트의 **End-to-End 프로세스**를 두루 경험한 엔지니어입니다. 특히 NLP(자연어 처리)와 추천 시스템 분야에서 강점을 보이며, 정형/비정형 데이터를 가리지 않는 데이터 핸들링 능력이 돋보입니다.

---

### 1. 프로젝트 개요
지원자는 총 4개의 주요 프로젝트를 통해 기술적 역량을 증명했습니다.
*   **북추 (도서 추천 서비스):** 사용자 맞춤형 연계 독서를 위한 3가지 방식의 추천 엔진 개발.
*   **음성 특징 구분 모델:** 사람의 음성 데이터를 분석해 성별과 연령대를 예측하는 CNN 모델 개발.
*   **허위소문 (뉴스 신뢰도 분석):** 낚시성 기사 및 뉴스 기사의 신뢰도를 분석하는 이진 분류 모델 개발.
*   **국민청원 카테고리 재분류:** 정책 변화에 따른 기존 청원 데이터의 다중 클래스 분류 모델 개발.

### 2. 사용 기술 (Tech Stack)
*   **Language & Data:** Python, SQL, Pandas, Numpy, BeautifulSoup (Web Crawling)
*   **Machine Learning:** Scikit-Learn (KNN, LDA, Matrix Factorization)
*   **Deep Learning:** Tensorflow, Keras, PyTorch
*   **NLP:** KoNLPy, Transformer (BERT, S-BERT, SKT-BERT), LLM API (GPT 2.4 Turbo)
*   **Audio/Vision:** Librosa, OpenCV

### 3. 기여도 (Contribution)
*   **데이터 파이프라인 구축:** 알라딘, 네이버 API 및 웹 크롤링을 통해 10만 건 이상의 도서 데이터를 직접 수집하고, 부족한 데이터를 보충하는 전 과정을 주도함.
*   **추천 알고리즘 다각화:** 단순한 협업 필터링을 넘어 텍스트 기반(LDA, S-BERT)과 사용자 기반(Matrix Factorization), 도메인 지식 기반(KNN)의 앙상블 접근 방식을 취함.
*   **가상 데이터 생성 및 검증:** 서비스 초기 데이터 부족 문제를 해결하기 위해 3,000명의 가상 사용자와 10만 건의 평점 데이터를 생성하여 모델을 검증하는 적극성을 보임.

### 4. 문제 해결 (Problem Solving)
*   **비정형 데이터의 특징 추출:** 음성 데이터를 직접 분석하기보다 **Librosa를 활용해 Spectrogram 이미지로 변환**하여 CNN 모델에 적용하는 방식을 선택함으로써, 복잡한 신호 처리 문제를 이미지 분류 문제로 치환하여 해결함.
*   **LLM의 전략적 활용:** 사용자의 도서 후기에서 키워드를 추출할 때 **GPT API를 활용해 해시태그 형식으로 요약**하고, 이를 S-BERT 임베딩으로 연결하여 의미론적 유사도를 계산하는 하이브리드 방식을 적용함.
*   **신뢰도 수치화:** 낚시성 기사 판별 시 단순히 결과만 내놓는 것이 아니라, **웹 크롤링과 연동해 실시간 기사를 분석하고 확률(퍼센테이지)로 출력**하여 사용자 경험을 고려함.

### 5. 기술적 숙련도 (Technical Proficiency)
*   **데이터 분석 기반(High):** 빅데이터분석기사, SQLD, ADsP 등의 자격증을 통해 증명된 탄탄한 이론적 기초를 갖추고 있음.
*   **NLP 모델링(Advanced):** Transformer 계열 모델(BERT, S-BERT)의 활용 능력이 우수하며, KoNLPy를 통한 한국어 형태소 분석 및 텍스트 정제 숙련도가 높음.
*   **추천 시스템 이해도(Advanced):** Matrix Factorization과 같은 고전적인 방식부터 최근의 임베딩 기반 유사도 측정까지 폭넓은 추천 엔진 메커니즘을 이해하고 있음.
*   **실행력 및 완성도:** 모든 프로젝트에서 ROC-AUC 0.9 이상 또는 만족도 95% 달성 등 정량적인 성과를 도출함.
